关于我们

我们都是学生，我们来自：

美国哈佛大学 / 美国卫斯理女子学院 / 美国宾夕法尼亚大学 / 美国康奈尔大学 / 美国康奈尔学院 /美国加州大学洛杉矶分校 / 美国北卡教堂山大学 / 美国欧柏林学院 / 美国汉密尔顿学院 / 美国密歇根大学 / 英国帝国理工大学 / 英国剑桥大学 / 英国牛津大学 / 英国纽卡斯尔大学 / 英国利物浦大学 / 加拿大卡尔顿大学 / 澳洲国立大学 / 美国曼荷莲女子学院 等

以及

美国、英国、加拿大、澳大利亚高中生联合开设的高中组 特设的法语组

我们的共同点：我们都是爱心论文计划的志愿者！

我们都是在读的高中生、本科生、硕士生和博士生，利用自己多年在海外教育系统的求学经验，为同学们一对一进行帮助，免费提供：

一篇或数篇论文修改和学术讨论 线上一对一环节，对自己学校的介绍、求学环境介绍、解答同学们的问题 同学们可以从以上两种活动中选择你想参加的活动，但是我们希望你为我们支持的公益组织项目提供一定数额的捐款（捐款详情请点击网站我们支持的公益组织页面进行详细了解）

我们可以辅导的论文囊括：

*GRE / GMAT / IELTS / TOEFL / SAT / CET4、6 *

高中论文 / 日常练习论文 / 法语练习论文

想参加？服务使用流程如下：

阅读爱心论文计划简介

根据组别分类浏览志愿者资料

阅读参与者须知并同意相关条款

填写修改论文申请并选择服务(EO/AS)

1）Essay Only: 志愿者为你修改文章，修改之后通过邮件返还；

2）Appointment Session:与志愿者亲切沟通留学生活、讨论择校问题、以及brainstorm你的essay！

为慈善事业贡献你的一份爱心
